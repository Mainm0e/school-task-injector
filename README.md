# Injector

This program will bind 2 program together

## Instruction

[install Go](https://go.dev/doc/install)

create 2 important files 
```
# Mac
touch important1.exe important2.exe

# Windows
echo. > important1.exe
echo. > important2.exe
```

## Usage

```
go run injector.go <file1> <file1>
```
